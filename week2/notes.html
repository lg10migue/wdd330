Ch2: Programming basics.

Comments

Comments are essential to make our programs easy to read and understand.

Primitive Data Types

There are six primitive data types.

  • String
  • Symbol
  • Number
  • Boolean
  • Undefined
  • Null

Template Literals

Use the ` to deliminate the string. They also allow interpolation of JavaScript code.

Non-primitive Data Type

Non-primitive data types, such as arrays, functions and objects, all have a type of "object".

Variables

Variables must be assigned before being used. Variables point to values stored in memory and are declared using the "const", "let" or "var" keywords. It is important to give variables sensible names that describe what they mean. JavaScript uses the camel-case notation to name variables and is the most recommended way to go, however, the most important thing is to be consistent in the method you use.

Properties and Methods

Strings and Numbers have various properties and methods that provide information about them or perform action on the object or value. For example: trim() method will remove any whitespace from te beginning and end of a string.

Symbols

Symbols are unique, inmutable values that helps to avoid naming collisions.

Boolean

Values that can be "true" or "false".

Falsy Values

JavaScript have nine values that are false.

Type Coercion

This is when JavaScript convert a value into another data type to perform an operation.

Ch3: Arrays, Logic, and Loops.

Arrays

Arrays are ordered list of values or objects. They have a lot of methods that can be used to manipulate them.

Multidimensional Arrays

Multidimensional arrays are arrays that contain other arrays.

Sets

Set are ordered lists of non-duplicate values.

Maps

Maps are ordered lists of key-value pairs.

Logic

We can use conditionals ( if>, else> ) to control the flow of code.

The switch statement can be used to avoid the use of a lot if and else statements.

Loops

A while loop and do ... while loop, can be used to repeat a block of code while a condition is still true.

A for loop, have different sintax of a while loop but works in similar way.

A for-of loop, can be use to iterate over arrays, sets and maps.

Ch4: Functions.

Functions

Functions are small parts of code that can be referenced by a name. Functions help to reduce repetition and make code easier to follow.

DRY

Stands for "Don't Repeat Yourself", is a principle of programming that specifies that every part of a program should only be written once. This avoids duplication and means there's no need to keep multiple pieces of code up to date and in sync.

Return Values

All functions return a value. If this is not explicitly stated, the function will return undefined.

Parameters and Arguments

A parameter is a value that is written in the function declaration and can be used like a variable inside the function's body.

An argument is a value that is provide to a function when it is invoked.

Default arguments can be supplied to a function by assigning them to the parameters.

Callbacks

A callback is a function that is provided as an argument to another function.
