Object Methods: "this".

Objects

Objects are created to represent entities of the real world. Actions are represented by functions in properties called methods.

Method Examples

We can allow or teach objects to "act" with methods ( object.doSomething() ). There are some ways to do this, for example, we can use Function Expression, Pre-declared Functions and Method Shorthand.

"this" in methods

To access the object, a method can use the this keyword. The value of this is the object "before dot", the one used to call the method.

"this" is not bound

The keyword this can be used in any function, eve if it is not a method of an object. The value of this is defined at run-time.

"this" in JavaScript.

This in a Global Context

When this is called outside any object, it is in global context, so this defaults to the Window object in the browser.

This in Object Construction

When we create a new instance of an object with the new keyword, this refers to the instance.

This in an Object Method

this within any method refers to the object itself. Since this refers to the object, we can use methods to get instance of an object.

This in a Simple Function

On browsers, this is always set to Window in a simple function. Even if we call a simple function in an object method.

This in Arrow Functions

this in an arrow function is always the same as this around it. If we use arrow functions, within an object method, the this context stays as the object.

This in Event Listeners

this is set to the HTML element that fired the event in an event listener.

Ch5: Objects.

Objects Literals

An object literal is an object that is created directly in the language by wrapping all its properties and methods in curly braces {}. Nested objects can be created by placing objects inside objects.

Objects are a collection of key-value pairs inside curly braces {}. Objects have properties that can be any JavaScript value. If it is a function, it is known as a method.

ES6 provide a Shorthand method of creating objects if a property key is the same as a variable name that the property value is assigned to.

Accesing Properties

An object's properties and methods can be accessed using either dot notation or square bracket notation.

Adding Properties

Objects are mutable, which means their properties and methods can be changed or removed.

Objects as Parameters to Functions

Objects can be used as parameters to functions, which allows arguments to be entered in any order, or omitted.

Namespacing

A namespace is a object literal pattern for groups of related functions.

JSON

JSON is a portable data format that uses string representation of JavaScript object literals to exchange information. A notable difference between them is JSON property names must be double-quoted.

The Math Object

The Math object gives access to a number of mathematical constants. It is used to perform mathematical operations.

The Date Object

The Date object can be used to create date objects. Setter methods can be used to change the information about that date.

The RegExp Object

The Regex object can be used to create regular expressions.

Special Characters:

  • . matches any character, except line breaks.
  • \w matches any word character, and is equivalent to [A-Za-z0-9_].
  • \W matches any non-word character, and is equivalent to [\^A-Za-z0-9_].
  • \d matches any digit character, and is equivalent to [0-9].
  • \D matches any non-digit character, and is equivalent to [^0-9].
  • \s matches any whitespace character, and is equivalent to [ \t\r\n\f].
  • \S matches any non-whitespace character, and is equivalent to [^ \t\r\n\f].

Modifiers:

  • ? makes the preceding token in the regular expression optional.
  • * matches one or more occurrences of the preceding token.
  • + matches one or more occurrences of the preceding token.
  • {n} matches n occurrences of the preceding token.
  • {n,} matches at least n occurrences of the pattern.
  • {,m} matches at most m occurrences of the preceding token.
  • {n,m} matches at least n and at most m occurrences of the preceding token.
  • ^ marks the position immediately before the first character in the string.
  • $ marks the position immediately after the last character in the string.

Ch6: The Document Object Model.

What is the DOM?

The Document Object Model is a way of representing a page of HTML as a tree of nodes.

Getting Elements

The DOM provides several methods to acces any element on a page.

All nodes have a numberical code to signify what type they are. These are summarized in the next table:

Code Type
1 element
2 attribute
3 text
8 comment
9 body

The document.getElementById(), document.getElementsByClassName(), document.getElementsByTagNames() and document.querySelector() can be used to access elements on a page.

Navigating the DOM Tree

The parentNode(), previousSibling(), nextSibling(), childNodes() and children() methods can be used to navigate around the DOM tree.

Getting and Setting Attributes

An element’s attributes can be accessed using the getAttribute() method, and updated using the setAttribute() method.

Classes Of An Element

.className

Using .className property allows the class of an element to be set directly.

.classList

The .classList property is a list of all the classes an element has. It has methods that make easier to modify the class of an element.

The add() method can be used to add a class to an element whitout overwriting any classes that already exists.

The remove() method will remove a specific class from an element.

The toggle() method will add a class if an element doesn't have it already, and remove the class if it does have it.

The contains() method will check if an element has a particular class.

Creating Dynamic Markup

The createElement() and createTextNode() methods can be used to create dynamic markup on the fly.

Markup can be added to the page using the appendChild() and insertBefore() methods.

Elements can be replaced using the replaceChild() method, and removed using the removeChild() method.

The innerHTML property can be used to insert raw HTML directly into the DOM.

Updating CSS

The CSS properties of an element can be changed by accessing the style property.
